# Who Should Use Biograph

Biograph is designed for a wide range of healthcare organizations and professionals:

| User Type | How They Use Biograph |
|-----------|----------------------|
| **Clinic Administrators** | Configure the system, manage users, set up departments and service units |
| **Front Desk / Reception** | Register patients, book appointments, handle check-ins |
| **Physicians / Practitioners** | Conduct consultations, record diagnoses, prescribe medications and tests |
| **Nurses** | Manage nursing tasks, administer medications, record vitals |
| **Lab Technicians** | Process lab orders, record results, generate diagnostic reports |
| **Pharmacists** | Dispense medications, manage stock, process prescriptions |
| **Billing Staff** | Generate invoices, process payments, manage insurance claims |
| **Hospital Management** | Monitor KPIs, review reports, oversee operations |
| **Patients** | Access the patient portal to view appointments, prescriptions, and lab results |
