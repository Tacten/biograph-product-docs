# Sample Collection

The **Sample Collection** workflow tracks specimens from the moment they are collected to when they reach the lab.

## Sample Collection Process

1. A **Lab Test** is ordered (from an encounter or directly)
2. A **Sample Collection** record is created (can be automatic if configured)
3. The phlebotomist or nurse:
   - Verifies patient identity
   - Collects the required sample
   - Labels the specimen with a barcode/ID
   - Records the collection time
4. The sample is transported to the laboratory
5. The sample status is updated as it progresses

## Sample Types

Pre-configure the sample types your lab works with:

| Sample Type | Common Tests |
|-------------|-------------|
| **Blood (Venous)** | CBC, Lipid Profile, Blood Glucose, Liver Function |
| **Blood (Arterial)** | Arterial Blood Gas (ABG) |
| **Urine** | Urinalysis, Urine Culture |
| **Stool** | Stool Routine, Occult Blood |
| **Sputum** | Sputum Culture, AFB |
| **Swab** | Wound Culture, Throat Culture |
| **CSF** | CSF Analysis |

## Barcode Support

Biograph supports barcode generation for specimen tracking:
- Each sample collection record can generate a unique barcode
- Barcodes can be printed on specimen labels
- Lab technicians can scan barcodes to quickly locate the corresponding lab test
