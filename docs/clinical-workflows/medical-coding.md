# Medical Coding

Biograph supports standardized medical coding for diagnoses and procedures:

## ICD-10

- **International Classification of Diseases, 10th Revision**
- Used for diagnosis coding worldwide
- Codes can be linked to diagnosis entries in the encounter
- Supports search by code or description

## SNOMED CT

- **Systematized Nomenclature of Medicine — Clinical Terms**
- Comprehensive clinical terminology
- Can be used alongside or instead of ICD-10
- Provides more granular coding options

## Code System Configuration

Medical codes are managed through:

- **Code System** — Define which coding systems you use (ICD-10, SNOMED, custom)
- **Code Value** — Individual codes within a system
- **Codification Table** — Mapping tables between different code systems
