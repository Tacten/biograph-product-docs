# Appointment Prerequisites

Before you can book appointments, ensure the following are in place:

| Prerequisite | Where to Set Up |
|-------------|----------------|
| **Healthcare Practitioner** | Healthcare Practitioner list — at least one practitioner must exist |
| **Practitioner Schedule** | Practitioner Schedule list — defines available time slots |
| **Appointment Type** | Appointment Type list — at least one type configured |
| **Healthcare Service Unit** | Healthcare Service Unit list — optional, for location-specific booking |
| **Patient Record** | Patient list — the patient must be registered |

> **Quick check:** If you can't see any available slots while booking, verify that the practitioner has an active schedule with time slots defined.
