# Nursing Checklist Templates

**Nursing Checklist Templates** define standardized sets of tasks for common care scenarios.

## Creating a Template

1. Go to **Nursing Checklist Template** list
2. Click **+ Add Nursing Checklist Template**
3. Give the template a name (e.g., "Pre-Surgery Checklist", "Admission Checklist", "Discharge Checklist")
4. Add tasks to the template:

| Task Field | Description |
|-----------|-------------|
| **Task Description** | What needs to be done |
| **Mandatory** | Whether this task must be completed |
| **Sequence** | Order in which tasks should be performed |

## Common Templates

| Template | Example Tasks |
|----------|--------------|
| **Admission Checklist** | Verify ID, Record vitals, Assign bed, Orient patient, Document allergies |
| **Pre-Surgery Checklist** | Confirm consent, NPO status, Mark surgical site, Confirm blood type, Remove jewelry |
| **Discharge Checklist** | Medication instructions, Follow-up appointment, Wound care instructions, Return belongings |
| **Shift Handover** | Review patient status, Pending medications, Pending results, Special instructions |

> **Tip:** Using templates ensures no steps are missed and maintains consistent care quality across nursing shifts.
