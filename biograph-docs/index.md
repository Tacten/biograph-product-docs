# Biograph Documentation

> Modern, open-source Healthcare Information System built on Frappe & ERPNext

Biograph is a comprehensive **Healthcare Information System (HIS)** designed to digitize and streamline healthcare operations for clinics, hospitals, and multi-specialty healthcare facilities. Built on the proven Frappe and ERPNext platform, Biograph provides a complete suite of tools that covers the entire patient care journey.
