# Who Should Use Biograph

Biograph is designed for a wide range of healthcare organizations and professionals:

| User Type | How They Use Biograph |
|-----------|----------------------|
| **[Clinic Administrators](../user-personas/clinical-administrator.md)** | Configure the system, manage users, set up departments and service units |
| **[Front Desk / Reception](../user-personas/reception--front-desk.md)** | Register patients, book appointments, handle check-ins |
| **[Physicians / Practitioners](../user-personas/physicians-practitioner.md)** | Conduct consultations, record diagnoses, prescribe medications and tests |
| **[Nurses](../user-personas/nurse.md)** | Manage nursing tasks, administer medications, record vitals |
| **[Lab Technicians](../user-personas/laboratory-user.md)** | Process lab orders, record results, generate diagnostic reports |
| **[Pharmacists](../user-personas/pharmacist.md)** | Dispense medications, manage stock, process prescriptions |
| **[Billing Staff](../user-personas/billing-staff.md)** | Generate invoices, process payments, manage insurance claims |
| **[Hospital Management](../user-personas/hospital--management-administartor.md)** | Monitor KPIs, review reports, oversee operations |
| **[Patients](../user-personas/patients.md)** | Access the patient portal to view appointments, prescriptions, and lab results |
